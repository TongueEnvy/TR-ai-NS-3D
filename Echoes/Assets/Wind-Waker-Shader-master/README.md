# Wind Waker Shader

Cel Shading of two blurred thresholds.

Author: Alberto Mellado Cruz
Date: 22/02/2018

This is a Cel Shading of 2 thresholds with a blur between them. 
It's intended to look like the shading in Wind Waker for GameCube.
